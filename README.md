# Iris Species Classifier (4-Class Version)

This repository contains a machine learning pipeline for 4-class Iris classification.

## 🚀 Key Features
- **4 Species Classification:** Includes Setosa, Versicolor, Virginica, and the custom **Aviv Iris**.
- **4x4 Confusion Matrix:** Visual performance tracking across all 4 categories.
- **97.5% Accuracy:** High-precision neural network implementation.
